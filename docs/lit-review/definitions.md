# Definitions

**Proxy communication**: AI-mediated communication where the AI system carries,
reshapes, times, filters, or partially authors a message between people.

**Communication boundary**: The point where one person's intent becomes another
person's received interpretation. A proxy system matters when it changes that
boundary.

**Delegated voice**: A case where a person lets an AI system speak on their
behalf, such as drafting a reply, summarizing a position, or negotiating a
handoff.

Boundary note: the draft should not use "proxy" for every AI writing aid. The
term is reserved for interpersonal communication surfaces.
