# Proxy Mechanisms

Candidate mechanisms:

- **Summarizing**: AI condenses another person's message or meeting record.
- **Filtering**: AI decides which messages, requests, or conflicts should be
  surfaced.
- **Rephrasing**: AI changes tone, formality, or cultural framing.
- **Delegating**: AI drafts or sends communication on behalf of a person.
- **Coordinating**: AI recommends when people should synchronize directly.

Evidence still needs to be tied back to the literature matrix before these
mechanisms are treated as final claims.
